Sweet Crumbs Cafe Website
A responsive website for Sweet Crumbs Cafe, developed as part of Web Programming Assignment 1.

Project Overview
Sweet Crumbs Cafe is a fully responsive website that showcases a local cafe's online presence. The website features multiple pages to provide customers with information about the cafe, its menu, story, and gallery.
